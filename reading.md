# Reading & Citations

This page highlights books and resources I reference in my work and study. Each item includes a link so you can explore the original source.

## Books I’m Reading or Citing

- **"Design Patterns: Elements of Reusable Object-Oriented Software"** by Erich Gamma, Richard Helm, Ralph Johnson, and John Vlissides
  - [Read more on Amazon](https://www.amazon.com/Design-Patterns-Elements-Reusable-Object-Oriented/dp/0201633612)

- **"Clean Code: A Handbook of Agile Software Craftsmanship"** by Robert C. Martin
  - [Read more on Amazon](https://www.amazon.com/Clean-Code-Handbook-Software-Craftsmanship/dp/0132350882)

- **"Deep Learning"** by Ian Goodfellow, Yoshua Bengio, and Aaron Courville
  - [Read more on the MIT Press site](https://www.deeplearningbook.org/)

- **"The Algorithm Design Manual"** by Steven S. Skiena
  - [Read more on the Springer site](https://www.springer.com/gp/book/9783030542553)

- **"Clean Architecture: A Craftsman's Guide to Software Structure and Design"** by Robert C. Martin
  - [Read more on ACM Digital Library](https://dl.acm.org/doi/10.5555/3175742)

- **"Unit Testing Principles, Practices, and Patterns"**
  - [Read more on IEEE Xplore](https://ieeexplore.ieee.org/book/10280451)

## Citation Notes

These books are useful for:

- Software architecture and design patterns
- Writing maintainable, production-ready code
- Understanding advanced machine learning concepts
- Designing efficient algorithms and data structures
